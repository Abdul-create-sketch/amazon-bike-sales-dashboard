# Amazon Bike Sales Dashboard 🚴‍♂️📊

## 📌 Project Overview
This project is a data analysis and visualization dashboard for bike sales data. It focuses on understanding customer behavior, sales trends, and key business insights using structured data.

## 📊 Project Name
Amazon Bike Sales Dashboard

## 🎯 Objectives
- Analyze bike purchase patterns
- Identify key customer demographics
- Understand factors influencing bike purchases
- Build interactive insights for decision making

## 🧰 Tools Used
- Microsoft Excel 
- Data cleaning & transformation
- Charts and dashboards

## 📈 Key Insights
- Sales distribution by gender, age, and income
- Customer commute distance impact on purchases
- Relationship between education and bike buying behavior
- Regional sales performance

## 📂 Dataset
The dataset contains customer demographic information such as:
- Age
- Gender
- Income
- Commute distance
- Education
- Marital status
- Bike purchase status

## 🚀 Outcome
A clear and interactive dashboard that helps understand bike sales patterns and supports business decision-making.
